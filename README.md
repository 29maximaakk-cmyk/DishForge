DishForge — Генератор случайных блюд на 7 языках
DishForge — это коллекция из семи независимых реализаций генератора случайных блюд, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для поиска кулинарных идей с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
🍽️ Обширная база рецептов: более 100 блюд с описанием, ингредиентами, временем приготовления и сложностью.

🎲 Генерация случайных блюд по различным критериям:

По типу (завтрак, обед, ужин, десерт, закуска, напиток)

По кухне (итальянская, японская, русская, мексиканская, французская, китайская, индийская)

По времени приготовления (до 15 мин, до 30 мин, до 1 часа, более 1 часа)

По сложности (лёгкое, среднее, сложное)

📊 Сохранение избранных рецептов в отдельный список.

📝 Экспорт рецепта в PDF или HTML (в некоторых версиях).

🌍 Поддержка нескольких языков (английский, русский, украинский — по выбору).

🖥️ Интерфейсы:

Командная строка (CLI) с гибкой фильтрацией.

Графический интерфейс (GUI) в Python, Java, C#.

Веб-интерфейс в PHP и JavaScript.

Использование (общий синтаксис)
CLI (все версии):

bash
dish generate [--type breakfast] [--cuisine italian] [--time 30] [--difficulty easy]
dish list
dish favorite add "Паста Карбонара"
dish random-menu --meals 3
Примеры:

Сгенерировать случайное блюдо:

bash
python dish_forge.py generate
Найти рецепт на завтрак за 15 минут:

bash
go run dish_forge.go generate -type breakfast -time 15
Сгенерировать меню из 3 блюд:

bash
node dish_forge.js random-menu --count 3
